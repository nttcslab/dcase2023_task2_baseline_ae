================================================================================
SYSTEM COMPARISON
================================================================================

System 1: BASELINE AE (MSE) - Clean Training
  - Training: 27 epochs (early stopped)
  - AUC (source/target/total): 0.7292 / 0.3694 / 0.5493
  - pAUC: 0.5358
  - Source F1: 0.5273, Target F1: 0.6622
  - Best val_loss: 8.9932 @ epoch 17

System 2: BASELINE + GLOBAL NORMALIZATION (BEST) ✓
  - Training: 50 epochs (full schedule with early stop at epoch 44)
  - AUC (source/target/total): 0.7550 / 0.3982 / 0.5766 (+2.8% source, +7.8% target)
  - pAUC: 0.5153
  - Source F1: 0.5275, Target F1: 0.6479
  - Best val_loss: 0.3738 @ epoch 44
  - Key Finding: Per-file amplitude normalization improved source-domain AUC
    by reducing feature-scale mismatch while maintaining training stability.

System 3: GLOBAL NORM + TEMPORAL STACKING
  - Training: 50 epochs (full schedule with early stop at epoch 44)
  - AUC (source/target/total): 0.5198 / 0.5376 / 0.5287 (-31% source vs System 2)
  - pAUC: 0.5537 (highest pAUC, but poor on source domain)
  - Source F1: 0.0000, Target F1: 0.0000 (decision threshold collapsed)
  - Best val_loss: 0.3138 @ epoch 44 (best reconstruction loss!)
  - Key Finding: While 5-frame temporal context improved target-domain ranking
    (ROC AUC 0.5376), it destabilized source-domain precision/recall and made
    the anomaly threshold less discriminative (F1 → 0).

================================================================================
THESIS CONCLUSIONS
================================================================================

1. SIMPLE IMPROVEMENTS WORK BEST
   Global normalization achieved the best overall AUC (0.5766) with a simple,
   reproducible preprocessing step. It improved both source and target domains
   without requiring architectural changes.

2. DOMAIN GAP REMAINS CHALLENGING
   Even with normalization, target-domain AUC (0.3982) shows a significant gap
   vs. source (0.7550). This reflects the inherent difficulty of cross-domain
   anomaly detection in the DCASE 2025 setting.

3. TEMPORAL CONTEXT IS DOUBLE-EDGED
   Temporal stacking captures local patterns but requires careful tuning.
   In this case, the 5-frame context improved target-domain ranking but
   broke source-domain binary classification due to instability in the
   decision threshold.

4. EARLY STOPPING IS CRITICAL
   Baseline: Converged in 27 epochs
   Globalnorm: Converged at epoch 44 (validation loss plateaued after that)
   Temporal: Plateau at epoch 44 despite different loss trajectory
   All systems benefited from monitoring validation loss.
